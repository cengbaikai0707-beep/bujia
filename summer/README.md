# 夏日英雄大會師・數學總複習

國小二～四年級數學核心概念整合複習，以三、四年級為主。**概念為主、計算為輔**——很多題答對之後還會追問一句「為什麼？」，理由選錯就不算真的懂。

由 `../hub.html` 進入，或直接開 `summer/index.html`。

## 檔案

| 檔案 | 內容 |
|---|---|
| `questions.js` | 題庫（`SUMMER_BANK`），純資料。 |
| `script.js` | 引擎：英雄技能、試煉場解鎖、四種題型判定、弱點追蹤、結算。 |
| `index.html` / `style.css` | 畫面與樣式（夏日海灘配色）。 |

## 六座試煉場

| 場 | 名稱 | 概念 |
|---|---|---|
| 1 | 數的王國 | 位值、0 的佔位、數感估算、數列與圖形規律 |
| 2 | 運算意義島 | 等號的意義、乘除的情境意義、等分除與包含除、餘數處理、運算順序 |
| 3 | 分數綠洲 | 等分割的必要、單位分數大小、整體不同不能比、等值分數、小數位值 |
| 4 | 測量高塔 | 時刻與時間量、角與邊長無關、長度重量容量換算、六十進位、量感 |
| 5 | 形狀神殿 | 周長與面積是不同的量、面積的意義與單位、圖形判準、對稱、方位 |
| 6 | 資料燈塔 | 圖表判讀與刻度陷阱、能與不能推論、平均的意義、數量關係 |

第一場預設開放，通過（正確率 60%）後解鎖下一場。六場全破後開啟**大會師總決賽**。

## 題型

| 類型 | 說明 |
|---|---|
| `reason` | **兩段式**：先選答案，答對再選「為什麼」。理由錯＝不算答對。這是本模組的主力題型。 |
| `multi` | 多選，選出所有正確說法。 |
| `order` | 依序點選完成排序。 |
| `single` | 單選。 |

## 英雄技能

進入試煉前必須先選一位英雄。每場可用一次技能：烈陽劍士（下一次答錯可重答）、浪花法師（刪去一個錯選項）、微風遊俠（看重點提示）、貝殼盾衛（下一次答錯不中斷連擊）。技能不會直接給答案。

## 弱點追蹤與總決賽

答錯會把該題的 `concept` 記進弱點紀錄（首頁可看），**答對同概念會自動減回去**。
總決賽優先抽出你答錯過的概念來補考——等於個人化的錯題複習。

## 資料維護

新增題目照現有欄位寫即可，`area` 與 `concept` 是動態彙整的，**新增領域不必改程式**。
`myth` 欄位（常見迷思）會在答錯時顯示並列進結算，建議每題都寫。

# 永齡學習偵探社・探索世界重構版

臺灣國小適用的純前端學習遊戲集合。免安裝、無後端、無帳號伺服器、無外部框架，可直接部署到 GitHub Pages。

## 正式入口

- 根目錄 `index.html`：GitHub Pages 預設入口。
- `hub.html`：舊網址相容用的轉址頁，會自動跳到 `index.html`（不再是重複的完整頁面）。
- 所有子模組的大廳連結統一回到 `../index.html`，避免 404。

## 七個免費核心館別

| 入口 | 資料夾 | 可取得證物 |
|---|---|---|
| 夏日英雄大會師 | `summer/` | 英雄徽章 |
| 部件偵探社 | `chinese/` | 文字證物 |
| 數感偵探社 | `math/` | 數字密碼 |
| 讀題偵探社 | `reading/` | 證詞紀錄 |
| 零食大盜 | `party/` | 合作線索 |
| 部件大亂鬥 | `battle/` | 怪物核心 |
| 部首字族失竊案 | `radical-case/` | 字族檔案 |

核心學習內容全部開放，不需要偵探幣解鎖。

## 共用探索系統

- `detective-system.js`：多個個人／小隊檔案、偵探幣、證據、經驗、背包、稱號與每日委託。
- 每次完整完成一館才結算，不以逐題亂點刷幣。
- 答錯不扣幣，會轉成「迷思通緝」，供之後的個人化救援案件使用。
- 商店幣可以買實際可用的排除選項、重新判斷、考點掃描、下一場補給與支線鑰匙。
- `guild/` 是選擇性的聯合案件局，包含封存案件、迷思救援與七館聯合案。
- 瀏覽器可保存多個檔案，也可匯出／匯入 JSON 跨裝置搬移進度。

## 部署

不要只上傳 ZIP，也不要再多包一層資料夾。

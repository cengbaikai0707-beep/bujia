# 國語學院學習大廳

純前端學習遊戲網站，不需要安裝套件。

```text
bujia-main/
├── index.html       學習大廳
├── hub.css          大廳樣式
├── chinese/         國文部件偵探社
├── math/            數學練習
├── reading/         讀題偵探社入口與應用題任務
├── radical-case/    中年級部首字族失竊案
└── party/           同樂會合作破案
```

直接開啟根目錄 `index.html`，或部署整個資料夾即可。

## 讀題偵探社

讀題偵探社目前有兩件案件：

1. 應用題審題診斷。
2. 中年級「部首字族失竊案」。

部首字族遊戲共98題、43組案件資料，完整模式抽30題，另有快速偵查、字量追緝與結案檢核。

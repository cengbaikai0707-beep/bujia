# 永齡學習偵探社・夥伴收藏完整版

臺灣國小適用的純前端學習遊戲集合。免安裝、無後端、無帳號伺服器、無外部框架，可直接部署到 GitHub Pages。

## 正式入口

- 根目錄 `index.html`：GitHub Pages 預設入口。
- `hub.html`：舊網址相容用的轉址頁，會自動跳到 `index.html`（不再是重複的完整頁面）。
- 所有子模組的大廳連結統一回到 `../index.html`，避免 404。

## 七個免費核心館別

| 入口 | 資料夾 | 可取得證物 |
|---|---|---|
| 夏日英雄大會師 | `summer/` | 英雄徽章 |
| 部件偵探社 | `chinese/` | 文字證物 |
| 數感偵探社 | `math/` | 數字密碼 |
| 讀題偵探社 | `reading/` | 證詞紀錄 |
| 零食大盜 | `party/` | 合作線索 |
| 部件大亂鬥 | `battle/` | 怪物核心 |
| 部首字族失竊案 | `radical-case/` | 字族檔案 |

核心學習內容全部開放，不需要偵探幣解鎖。

## 共用探索系統

- `detective-system.js`：多個個人／小隊檔案、偵探幣、證據、經驗、背包、稱號與每日委託。
- 每次完整完成一館才結算，不以逐題亂點刷幣。
- 答錯不扣幣，會轉成「迷思通緝」，供之後的個人化救援案件使用。
- 商店幣可以買實際可用的排除選項、重新判斷、考點掃描、下一場補給與支線鑰匙。
- `guild/` 是選擇性的聯合案件局，包含封存案件、迷思救援與七館聯合案。
- 瀏覽器可保存多個檔案，也可匯出／匯入 JSON 跨裝置搬移進度。

## 偵探夥伴養成

- `pet/` 是主要養成頁；題目任務是取得飼料、偵探幣、布置與進化材料的主要方式。
- 已領養的夥伴會成為全站隨行像素生物：在大廳與各答題館底部自行走動、休息，點擊可摸摸牠，並會依答對、答錯與任務完成做出不同反應。
- 九種動物素材都存放於 `assets/pets/`，不依賴外部圖片服務；手機版會縮小活動範圍，並尊重系統的減少動態效果設定。
- 初始可選線索犬、推理貓、跳跳兔、口袋鼠。
- 檔案鴞、墨燈狐、守護熊、整理企鵝、神祕幼龍需透過館別證物、跨館探索或聯合案件解鎖。
- 同一份偵探檔案可收藏多位夥伴並自由切換；每位夥伴各自保存名字、成長階段、親密度、房間及配件。
- 夥伴成長採彈性 2＋2＋3 路線：孵化可自選 2 館、少年期再從未使用的館中自選 2 館、成熟期完成剩下 3 館，每館各需 1 個材料。孩子能決定練習順序，但完整養成仍會接觸七館。
- 商店提供照顧用品、五種配件與四種房間背景，作為偵探幣的長期使用目標。
- 夥伴離線時只會緩慢降低狀態，最低停在安全值；不會因學生一週未登入而死亡或自動生病。
- 舊版單一夥伴存檔會自動搬進新版收藏。

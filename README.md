# 永齡學習偵探社・夏日英雄整合版

臺灣國小適用的純前端學習遊戲集合。免安裝、無外部框架，可直接部署到 GitHub Pages。

## 正式入口

- 根目錄 `index.html`：正式學習大廳，也是 GitHub Pages 網址的預設入口。
- `hub.html`：同一座學習大廳的備用網址。

所有子模組的「回學習大廳」都統一回到 `../index.html`，避免資料夾層級不同造成 404。

## 七個任務入口

| 入口 | 資料夾 | 內容 |
|---|---|---|
| 夏日英雄大會師 | `summer/` | 二～四年級數學核心概念、六場試煉、英雄技能與弱點補考 |
| 部件偵探社 | `chinese/` | 部件識字與字義推理 |
| 數感偵探社 | `math/` | 數感與個別補強 |
| 讀題偵探社 | `reading/` | 96 題自由練習＋中高年級兩階段診斷 |
| 零食大盜 | `party/` | 國文 × 數學 × 讀題合作案件 |
| 部件大亂鬥 | `battle/` | 部件延伸與 Boss 闖關 |
| 部首字族失竊案 | `radical-case/` | 部首字義、字族延伸與字量搜索 |

## 本版整合重點

- 夏日英雄改為大廳最上方的夏季主線入口。
- 根網址直接顯示學習大廳。
- 恢復新版讀題診斷與部首字族獨立模組。
- 修正烈陽劍士技能重複生效、浪花法師刪除顯示錯位。
- 選擇英雄後才能進入試煉。
- 全站本機連結與 JavaScript 語法均經檢查。

## 部署

解壓縮後，把本資料夾裡的全部檔案與資料夾上傳到 GitHub repository 根目錄。不要只上傳 ZIP，也不要再多包一層資料夾。

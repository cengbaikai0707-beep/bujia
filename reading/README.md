# 讀題偵探社

給國小中高年級練「讀懂數學應用題」的診斷型練習：不是練計算，而是從錯誤選項反推學生漏看了哪個關鍵條件。題庫 96 題，每題帶 `skill`（能力）與 `trap`（陷阱）標籤。

由 `../hub.html` 進入，或直接開 `reading/index.html`。

## 檔案

| 檔案 | 內容 |
|---|---|
| `questions.js` | 題庫 96 題（`READING_QUESTION_BANK`）。 |
| `detective-system.js` | 偵探系統核心：偵探幣、道具商店、信心押注、陷阱怪對照與圖鑑、復仇任務、每日事件。純邏輯，不碰 DOM。 |
| `script.js` | 遊戲流程與介面。 |
| `index.html` / `style.css` | 畫面與樣式。 |

## 偵探系統

**偵探幣**：答對即得，依難度給（基礎 8／標準 10／挑戰 14）。

**信心押注**（作答前選，練後設認知——分辨「真的懂」還是「憑感覺猜」）：

| 模式 | 答對 | 答錯 |
|---|---|---|
| 穩健調查 | 標準幣 | −3 幣 |
| 孤注一擲 | 2 倍幣 | −8 幣 |
| 購買保險 | 標準幣（先付 5） | 不扣幣 |

幣數不會低於 0；本模組**不使用生命值**，答錯不會提前結束練習，以保留完整診斷資料。

**道具商店**：🔍 放大鏡（排除一個錯誤選項）、🛡️ 冷靜卡（本題答錯不扣幣）、📜 偵探筆記（提前揭露本題陷阱怪）。

**陷阱怪**：題庫的 37 個 `trap` 標籤歸併為 8 隻怪（問句小偷、其中怪、剩下幽靈、比較忍者、干擾史萊姆、換算小妖、漏步蝸牛、至少最多獸），未對到的標籤歸「迷霧干擾怪」。結算不寫「你錯了第 3 題」，而是「遭到問句小偷攻擊 2 次」。

**復仇任務**：被同一隻怪累積攻擊 3 次後解鎖，抽 3 題同家族題目，全對可獲得稱號（如「問句小偷剋星」）與 30 幣。

**每日事件**：依日期輪替——平靜的一天／霧夜調查（幣加倍）／道具折扣日（半價）。

## 資料維護

- 新增題目只要照原本欄位寫；`skill` 與 `trap` 是動態彙整的，**新標籤不需改程式**。
- 新的 `trap` 標籤若想歸到指定陷阱怪，在 `detective-system.js` 的 `trapMap` 加一行即可；沒加也會安全落到「迷霧干擾怪」。

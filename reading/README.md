# 讀題偵探社 2.0

這個模組可以直接離線使用，不需要安裝套件或連接資料庫。

## 內容

- `index.html`：首頁、作答畫面與結果報告
- `script.js`：抽題、兩階段作答、計分與匯出
- `style.css`：版面與行動裝置樣式
- `questions.js`：原有 96 題自由練習題庫
- `diagnostic-questions.js`：新版分層診斷題庫

## 完整診斷

- 中年級、高年級各 12 題
- 六項能力各固定抽 2 題
- 每題分成「第 1 關：讀懂」與「第 2 關：算對」
- 支援一般文字、表格、價目、規則與時刻表
- 記錄線索使用情形，結果可匯出 JSON

六項能力為：問句定位、條件篩選、關係辨認、步驟排序、限制判斷、單位與表徵。

## 新增診斷題

請在 `diagnostic-questions.js` 的 `READING_DIAGNOSTIC_BANK` 陣列中，照現有格式新增。每個年級帶與每項能力至少保留 2 題，否則完整診斷無法維持固定取樣。

注意：這是教學用的形成性診斷。每項能力一次只有 2 次觀察，適合找練習方向，不等同正式心理或學力測驗。

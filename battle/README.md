# 部件大亂鬥・文字勇者

給國小三～六年級的部件識字闖關遊戲。核心不是背字，而是走完
**辨認部件 → 修復文字 → 理解字義 → 組成字族 → 放回語境驗證** 的推理歷程。

直接開啟 `../hub.html` 進入大廳，再點「部件大亂鬥」；或直接開 `battle/index.html`。

## 檔案

| 檔案 | 內容 |
|---|---|
| `component-data.js` | 部件教材資料（`COMPONENT_BANK`）：`comp` 顯示形、`base` 本形、`role` 形旁/聲旁/部件、`meaning`、`grade`、`level`、`examples`、`note`。例字皆經 IDS 字形分解驗證。 |
| `question-data.js` | 題目資料：`FAMILY_DATA` 聲旁字族白名單、`CONTEXT_DATA` 情境鑑識、`IMPOSTER_DATA` 冒牌字。皆為人工確認之存在字。 |
| `script.js` | 遊戲引擎：題型出題、三能力值、Boss、關卡解鎖、localStorage、教師模式。 |
| `index.html` / `style.css` | 畫面與樣式。 |

## 題型

辨認、部件意義、部件修復（組字＋字義兩階段）、字族連鎖、情境鑑識、冒牌字鑑識，
每關混合出題；約三～四成題目會追問「你根據哪一種線索判斷」。

## Boss

形近怪（變形→本形配對）、月肉雙面怪（分類）、左右阝迷宮（分類）、聲旁變色龍（依義配字）。
Boss 有獨立修復進度，答錯可重試、不會一次出局。

## 能力值

字形完整度、字義穩定度、語境清晰度三項，於結算分開顯示並指出最穩定／最需補強的能力。

## 進度

第一關預設開放，通過後解鎖下一關；Boss 通過後打通下一區域。進度存於 `localStorage`。
勾選「教師自由模式」可直接選任一關卡（方便課堂投影與指定練習）。

## 資料維護原則

- 不混入簡體字、不生成不存在的漢字組合；聲旁字族一律人工白名單。
- 區分本形與常見變形（如「礻」是「示」在左邊的變形）。
- 不確定的字源採安全說法，不武斷宣稱唯一造字來源。
